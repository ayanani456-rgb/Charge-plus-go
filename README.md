# CHARGE+GO

### Plug In. Power Up. Go.

Smart power bank rental kiosk for university students in Pakistan.

---

##  The Problem

No wall sockets + phones always dying + expensive power banks = students sitting on floors for 30+ minutes.

##  The Solution

Scan QR → Pay PKR 1,500 deposit → Take power bank → Use anywhere → Return to ANY kiosk → Get PKR 1,300 back

**You pay only PKR 200!**

---

##  Impact

| Metric | Value |
|--------|-------|
| Survey Responses | 122 students |
| Willing to pay PKR 100-200 | 86% |
